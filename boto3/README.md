# Boto3

### Installation
```
$ sudo apt install python-pip
```

```
$ pip install boto3
```

### Configuration
Configure credentials (in e.g. ~/.aws/credentials):
```
[default]
aws_access_key_id = YOUR_KEY
aws_secret_access_key = YOUR_SECRET
```

Then, set up a default region (in e.g. ~/.aws/config):
```
[default]
region=us-east-1
```
